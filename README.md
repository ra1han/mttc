# Hello World View Extension

A simple VS Code extension that displays a "Hello World" view in a webview panel.

## Features

This extension adds a single command that opens a webview panel displaying "Hello World" in large, centered text.

- Command: `Hello World` - Opens the Hello World view

## Usage

1. Open the Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P` on macOS)
2. Type "Hello World" and select the command
3. A new panel will open displaying "Hello World"

## Requirements

- VS Code version 1.106.0 or higher

## Development

### Setup

```bash
npm install
```

### Compile

```bash
npm run compile
```

### Run Extension

Press `F5` to open a new VS Code window with the extension loaded.
